# Blair Project - Claude Code Permissions

## Allowed Commands
- Run npm/node commands (npm run dev, npm run build, npm install, etc.)
- Run git commands (git status, git add, git commit, git push, git pull, etc.)
- Kill/restart dev servers (pkill, lsof)
- File system operations within this project

## Project Context
This is Blair, an AI fax sorter prototype for a cardiology clinic RFP.
- Next.js 15 + TypeScript + Tailwind + shadcn/ui
- Mock data only, no real backend
- Dev server typically runs on localhost:3000
